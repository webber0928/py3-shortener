# py3-URL-shortener

## create db

`python3 script/create_db.py`

## api list

```
1. check server work
[GET] /

Response:
    200:
        { "code": 200, "message": "ok." }
```

```
2. create short url
[POST] /

Body:
    {
        "origin": "https://www.google.com.tw?region=zh_tw#/firstpage",   # redirect url
	    "alias": "dcRi4m"                                                # (option)
    }

Response:
    200:
        { "code": 200, "message": "ok." }
    400:
        # origin is None
        { "code": 400, "message": "Validation Failed" }
    410:
        # alias repeat
        { "code": 410, "message": "Alias repeat" }
```
